Git Reset
----------

-> git reset 4deaf
  it takes us back .. but our working directory has not changed..
  so we can commit changes again..
  
  
-> git reset 4deaf --hard 
    this command also changes the working directory..

Git Revert
----------
-> git revert 29abce
    it reverts one particular commit
    
Git Checkout
------------
 used to see state of code in earlier commit..
